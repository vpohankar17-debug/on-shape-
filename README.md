# Flange Coupling Assembly

## Overview
This project presents the design and assembly of a Flange Coupling created using Onshape. The coupling is designed to connect two shafts for efficient torque transmission while maintaining alignment.

## Software Used
- Onshape
- Mechanical CAD

## Components
- Flange (2)
- Shaft
- Bolts
- Nuts

## Features
- Complete 3D assembly
- Parametric CAD model
- Standard mechanical dimensions
- Ready for manufacturing documentation

## Applications
- Power transmission systems
- Industrial machinery
- Pumps
- Compressors
- Gearbox assemblies
